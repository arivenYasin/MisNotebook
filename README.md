# 知识图错题本

一个运行在浏览器中的**个人知识图谱 + 间隔复习**错题本工具，可以便捷的导入错题，并根据错题关联的知识自动绘制知识图谱，还可以根据知识的掌握情况自动推荐复习题。

---

## 功能简介

- **知识图谱**：将知识点组织为树形结构，用 D3.js 渲染为可交互的力导向图。父子关系用渐变粗线连接，共享题目的知识点之间用虚线和方框徽章表示关联，点击徽章可展开为扇形题目球。
- **题目管理**：录入题目并关联一个或多个知识点，记录对错及失败知识点，支持在图谱中直接点击节点多选关联知识点。
- **掌握度/置信度可视化**：节点颜色实时反映掌握度（绿→黄→红），掌握度极低时有红色脉冲警告圈。
- **智能推荐复习**：混合 Leitner 记忆曲线、优先级公式和探测机制，每次生成一组推荐题目，带进度条。
- **题目详情弹窗**：点击题目小球可查看关联知识点、历史记录、直接作答并更新数据。
- **数据本地持久化**：所有数据保存在浏览器 `localStorage`，无需登录或联网，刷新不丢失。

## 快速开始

**环境要求**：Node.js 18+

```bash
# 克隆仓库
git clone https://github.com/arivenYasin/kgraph-review.git
cd kgraph-review

# 安装依赖
npm install

# 启动开发服务器（热更新）
npm run dev
```

浏览器打开 `http://localhost:5173` 即可使用。首次使用可点击顶部「◆ 加载示例数据」体验内置的考研数学示例。

### 构建为静态页面部署

```bash
npm run build
# 产物在 dist/ 目录，可直接托管到任意静态文件服务（GitHub Pages、Vercel、Nginx 等）
```

## 使用方法

1. **建立知识树**：点击左侧「**+ 知识点**」，先建章节（顶级节点），再逐层建子节点。
2. **录入题目**：点击「**+ 题目**」或在图上右键节点选「添加题目」，可点击「在图中选择」按钮后在图谱上点击多个节点关联多个知识点。
3. **浏览图谱**：双击节点展开/折叠子树；右键节点可显示该节点的题目小球；滚轮缩放，拖拽节点可重新布局。
4. **查看题目**：点击题目小球打开详情弹窗，可直接作答（系统将更新掌握度/置信度），也可删除题目。
5. **开始复习**：点击顶部「**▶ 开始复习**」，系统生成推荐题单，逐题作答并标记对错。

---

### 5. 参数速查表

所有参数集中在 `src/knowledge-graph-app.jsx` 顶部的 `CONFIG` 对象：

| 参数 | 变量名 | 含义 | 默认值 |
|------|--------|------|--------|
| λ | `lambda` | 置信度时间衰减基础系数 | 0.95 |
| η | `eta` | 置信度学习率 | 0.4 |
| k | `k` | 遗忘倾向对衰减速率的调节系数 | 2.5 |
| w | `w` | 优先级公式两项的平衡权重 | 4 |
| α | `sigmoidAlpha` | 掌握度 sigmoid 的陡峭程度 | 0.7 |
| — | `explorePct` | 探测题占非到期配额的比例 | 0.2 |
| — | `batchSize` | 每次复习生成的题目数 | 10 |
| — | `minInterval` | 同一题再次出现的最小间隔步数 | 3 |
| — | `leitnerIntervals` | Leitner 各盒的间隔步数 | [1,2,4,8,16,32] |

## 项目结构

```
src/
  knowledge-graph-app.jsx   # 全部逻辑（算法、D3 模拟、React 组件、样式），约 1900 行
  App.jsx                   # 入口包装（<KnowledgeGraphApp />）
  main.jsx                  # React 挂载点
requirements.md             # 原始需求文档，含完整算法设计意图
CLAUDE.md                   # AI 辅助开发说明文档
```

> 项目目前为单文件架构（所有逻辑在 `knowledge-graph-app.jsx`），便于快速迭代。如需大规模重构，建议先拆分为独立的 hooks、components、utils 目录。
